# iFitness Hub — PWA Deploy

## Files in this folder
- `index.html` — the app
- `manifest.json` — PWA manifest (makes it install as real app)
- `sw.js` — service worker (enables offline + install)
- `icon-192.jpg` — app icon
- `icon-512.jpg` — app icon (large)

## How to host FREE on GitHub Pages (5 minutes)

1. Go to https://github.com and sign in (or create free account)
2. Click **New repository** → name it `ifitness` → set to **Public** → click **Create**
3. Click **uploading an existing file** (or drag & drop)
4. Upload ALL 5 files from this folder
5. Click **Commit changes**
6. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)** → Save
7. Wait ~1 min → your app is live at: `https://YOUR-USERNAME.github.io/ifitness/`

## Install on Android (Chrome)
1. Open the URL above in Chrome
2. Tap the **INSTALL** button that appears in the app
3. Tap **Install** in Chrome's dialog
4. Done — real app icon, NO Chrome badge!

## Install on iPhone (Safari)
1. Open the URL in Safari
2. Tap **Share** → **Add to Home Screen** → **Add**
